# Maternal Health Monitor & Preeclampsia Predictor
### This project is a part of Equitable Innovation Programme conducted by Lehigh University, USA and KGiSL Institute of Technology, Coimbatore
This platform is a specialized clinical dashboard designed for the continuous monitoring of maternal patients. It integrates real-time health data with AI-driven predictive analytics to identify early warning signs of conditions like preeclampsia, enabling healthcare providers to intervene before critical complications arise.

## 🚀 Key Features
* Continuous Maternal Monitoring:
  * Live Vital Sign Tracking: Real-time visualization of blood pressure, heart rate, and other critical metrics essential for preeclampsia screening.
  * Nurse Desk Interface: A dedicated view for clinical staff to manage multiple maternal patients simultaneously and track daily progress.
* AI-Powered Risk Prediction:
  * Preeclampsia Risk Scoring: Automated assessment tools that process monitoring data to generate intuitive risk levels.
  * Risk Gauges: Visual indicators (High/Medium/Low) on patient profiles to highlight those requiring immediate attention.
*Clinical Data Management:
  * Longitudinal Patient Profiles: Comprehensive histories of vital signs and clinical notes to track health trends over the course of pregnancy.
  * Update Forms: Easy-to-use interfaces for clinicians to log new observations or adjust patient parameters.
* Intelligent Alerting:
  * Critical Alerts Panel: A high-priority section that flags abnormal vitals or AI-detected risks for the Emergency or ICU teams.
  * Real-Time Notifications: Instant alerts for lab results or significant changes in a patient's predictive risk score.

## 🛠 Tech Stack
- Frontend: React (TypeScript) + Vite.
_ Styling: Tailwind CSS and shadcn/ui for a clean, medical-grade interface.
- Database & Real-time: Supabase for secure, live data synchronization.
- Analytics: Recharts for visualizing maternal health trends and predictive data.
- State Management: Zustand for efficient handling of real-time monitoring streams.
